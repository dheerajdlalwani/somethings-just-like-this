4. I Tried To Be...

08/04/2022

I tried to be someone else's hero
Only to realize
I was the one
Who needed To be rescued.


I tried to be someone else's teacher
Only to realize
I was the student
In need of help.


I tried to be someone else's savior
Only to realize
I was the one
Who needed to be saved.


I tried to be someone else's close friend
Only to realize
I was my very own
Best friend forever.


I tried to be like someone else
Only to realize
I just had to be
Better version of my own self.


I tried to be someone else's something
Only to realize
I was my very own
Everything :)

- Dheeraj Lalwani
